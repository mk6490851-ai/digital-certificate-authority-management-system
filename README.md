# Web-Based Digital Certificate Authority Management System (WCAMAN)

## 📌 Project Overview
WCAMAN is a web-based Digital Certificate Authority (CA) Management System developed to issue, revoke, and renew SSL/TLS certificates securely.

## 🚀 Technologies Used
- Node.js
- Express.js
- MySQL
- Bootstrap
- Docker Engine (Linux)
- Step-CA
- Step-CLI

## 🔐 Key Features
- SSL Certificate Issuance
- Certificate Renewal
- Certificate Revocation
- Email OTP Authentication
- Certificate Expiration Email Notifications
- Dockerized Deployment

## 🐳 Docker Setup
1. Install Docker
2. Clone repository
3. Run:
   docker-compose up --build

## 📂 Installation
1. Install Node.js
2. Install dependencies:
   npm install
3. Configure .env file
4. Start server:
   npm start
